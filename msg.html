LLamasoft Message Exercise

Take The Challenge

Implement a messaging system to replace the random alert boxes that initially appear.

Feel free to use any third party libraries you find helpful. jQuery and underscore are already referenced.

Requirements

The messaging system should meet the following criteria:

  • Messages should appear near the top of the page
  • Messages should fade out after 3 seconds
  • If a new message appears before the previous message fades out, it should appear below the most recent visible message.
  • Messages should have a close or delete box to dismiss the message before the fade out.

Bonus

In addition to the above features, implement other updates you think might be helpful. E.g.:

  • Add a secondary indicator to let the user know that messages are running (in addition to the text change on the button)
  • Update the layout to make the page more readable, etc.
